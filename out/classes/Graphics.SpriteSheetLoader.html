Graphics.SpriteSheetLoader Class

Extends AbstractLoader
Defined in: Scripts\lib\PreloadJS\lib\preloadjs-NEXT.combined.js:6997

A loader for EaselJS SpriteSheets. Images inside the spritesheet definition are loaded before the loader completes. To load SpriteSheets using JSONP, specify a LoadItem/callback:property as part of the LoadItem. Note that the JSONLoader and JSONPLoader are higher priority loaders, so SpriteSheets must set the LoadItem LoadItem/type:property property to AbstractLoader/SPRITESHEET:property.

The LoadItem LoadItem/crossOrigin:property as well as the LoadQueue's basePath argument and LoadQueue/_preferXHR property supplied to the LoadQueue are passed on to the sub-manifest that loads the SpriteSheet images.

Note that the SpriteSheet JSON does not respect the LoadQueue/_preferXHR:property property, which should instead be determined by the presence of a LoadItem/callback:property property on the SpriteSheet load item. This is because the JSON loaded will have a different format depending on if it is loaded as JSON, so just changing preferXHR is not enough to change how it is loaded.
