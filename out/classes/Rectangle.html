Rectangle Class

Defined in: Scripts\lib\EaselJS\lib\easeljs-NEXT.combined.js:2398

Represents a rectangle as defined by the points (x, y) and (x+width, y+height).

Example

 var rect = new createjs.Rectangle(0, 0, 100, 100);
